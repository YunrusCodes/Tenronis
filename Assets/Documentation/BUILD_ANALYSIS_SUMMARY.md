## 🎯 核心問題

### 當前狀態：設計基準錯位

```
發現：遊戲設計隱含假設 Expert + Volley Build

證據：
1. Stage 20 HP = 5,000
   Expert+Volley單次爆發 = 13,440
   → 設計者期望2-3次消除（但實際只需2秒）

2. 射速加速 3.0s → 1.0s（3倍）
   → 假設長戰鬥時間（但Expert瞬間擊殺）

3. 後期子彈複雜度暴增（8種類型）
   → 假設玩家需要應對（但Expert擊殺太快）

結論：整個後期設計都以"極端高火力"為基準
```

### 對不同群體的影響

| 群體 | 當前體驗 | 問題 |
|------|---------|------|
| Expert + Volley | 2秒擊殺 | Boss無挑戰性 ✗ |
| Intermediate | 6-9秒擊殺 | 略顯無聊 ✗ |
| Casual | 38秒擊殺 | 壓力來源錯誤（射速而非HP）✗ |

**所有群體都不滿意！**

## 📊 數學證明：Volley是Expert專屬

### 公式推導

```
PDA_volley(tier) ∝ [λ_clear × E[r]] × (1+L_volley) × C(tier)

層級差異（Stage 20，L_volley=5）：
Expert:  [0.219] × 6 × 30 = 39.4 單位
Inter:   [0.200] × 6 × 10 = 12.0 單位
Casual:  [0.194] × 6 × 5  = 5.8 單位

比率：Expert : Inter : Casual = 6.8 : 2.1 : 1
```

**含義**：Volley收益隨技能層級呈**指數增長**

### 實際傷害對比（Stage 20）

```
Expert+Volley:  PDA = 2,943  T_kill = 2.0s
Inter+Volley:   PDA = 1,200  T_kill = 5.9s
Casual+Volley:  PDA = 235    T_kill = 38.3s

差距：Expert是Casual的 12.5倍！
```

**結論**：如果只看Volley平衡 → 必然以Expert為基準 → 其他層級完全失衡

## 🎯 三大核心結論

### 結論1：不能只用Volley看平衡

**原因1 - 數學**：
```
Volley收益 = 層級差距6.8:2.1:1
無法用單一HP滿足所有層級
```

**原因2 - 設計**：
```
當前隱含假設Expert+Volley
→ 其他情況全部失衡
```

**原因3 - 玩家**：
```
不同層級需要不同Build
單一視角無法滿足
```

---

### 結論2：Build路線的數學角色

#### Volley = 火力放大器
```
數學特性：線性乘數（每級+100%導彈）
適用層級：Expert（高Combo放大效果）
數學角色：主要輸出源

當前問題：無上限 → Expert過強
建議：上限L=4，或遞減收益（-28%）
```

#### Defense = 時間緩衝器
```
數學特性：方塊存活時間 = (1+L_defense)/λ_bullet
適用層級：Casual（低火力 = 長戰鬥）
數學角色：轉化時間為容錯空間

當前問題：無明確上限
建議：上限L=10（提供11秒緩衝）
```

#### Tactical = 非線性爆發源
```
數學特性：DPS有周期性波動（技能爆發）
適用層級：Intermediate（平衡取向）
數學角色：提供戰術節奏

當前狀態：良好（技能DPS佔30%）
建議：維持不變
```

---

### 結論3：需要三軌制平衡設計

**單一平衡 → 三軌平衡**

```
軌道1：輸出軌（Volley Build + 專家模式）
目標：Expert玩家
難度：HP × 7.0
目標時間：15-25秒

軌道2：坦克軌（Defense Build + 休閒模式）
目標：Casual玩家
難度：HP × 0.6
目標時間：30-50秒

軌道3：戰術軌（Tactical Build + 標準模式）
目標：Intermediate玩家
難度：HP × 1.8
目標時間：20-35秒
```

## 🎯 關鍵要點總結

### 為什麼不能只用Volley看平衡？

**三大原因**：
1. **數學原因**：收益呈指數增長（6.8:2.1:1）
2. **設計原因**：當前隱含假設導致失衡
3. **玩家原因**：不同層級需要不同Build

### Defense/Tactical的數學角色

**Defense = 時間緩衝器**
- 轉化長戰鬥時間為容錯空間
- 合理上限：L=10（11秒緩衝）

**Tactical = 非線性爆發源**
- 提供戰術節奏和靈活性
- 當前設計良好（技能DPS佔30%）

### 解決方案

**三軌制平衡設計**：
- Volley + 專家模式（Expert）
- Defense + 休閒模式（Casual）
- Tactical + 標準模式（Intermediate）

**預期效果**：
- 所有層級獲得適合的挑戰
- Build多樣性增加
- 重玩價值提升
